1 FastAPI là gì? 
ANS: fastAPI là web framework hiện đại để xây dựng REST API hiệu năng cao, dễ viết, dễ test và tự động sinh API docs

2 ASGI server là gì?
ASGI (Asynchronous Server Gateway Interface) là chuẩn giao tiếp giữa web server và Python app.
- Cho phép python xử lý dữ liệu bất đồng bộ
- Real time, websocket
- High concurrency, Long polling
